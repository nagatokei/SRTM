SRTM ユーザーガイド / README

このページでは、SRTM（Webカメラフェイストラッキングシステム）の詳細な設定やトラブルシューティングについて解説します。

⚙️ セットアップの詳細

基本操作はポータル画面の通りですが、より安定させるためには以下の設定を推奨します。

  • ブラウザのフォーカス: 送信機側は常に小さなウィンドウで表示させておくと、OSによるスリープを防げます。
  • カメラ権限: 起動時にカメラの使用を許可してください。

❓ よくある質問 (FAQ)

Q. アバターが動きません。

A. VRM1.0を選択し、送信機(Transmitter)の画面で送信開始をクリックしてみてください。また、受信機(Receiver)側で正しいPeerIDが入力されているかチェックしてください。

Q. 動作が非常に重いです。

A. ブラウザの「ハードウェアアクセラレーション」がオンになっているか確認してください（Chrome設定 ＞ システム）。

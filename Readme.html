SRTM ユーザーガイド / README

このページでは、SRTM（Webカメラフェイストラッキングシステム）の詳細な設定やトラブルシューティングについて解説します。

⚙️ フェイストラッキングについて

本システムはVRM1.0に対応しており、以下の仕様で動作します。

  • トラッキング項目: 目、鼻、口、頬の位置情報。
  • パーフェクトシンク: 対応モデルでは最大48か所のポイントを使用します。
  • 軽量簡易モード: ラグがある場合は「拡張表情モード」のチェックを外してください。
  • ブラウザのフォーカス: 送信機側を小さなウィンドウで表示させておくとスリープを防げます。
  • カメラ権限: 起動時にカメラの使用を許可してください。

💡 挑戦してみたい方へ
VRoid Studio 1.26.0以降（VRM 1.0形式）で作成されたモデルを [パーフェクトシンクブレンドシェイプリスト追加ツール] で読み込んでみてください。（リストに応じたブレンドシェイプの編集にはblenderやUnityが必要です。）

  • ブラウザのフォーカス: 送信機側は常に小さなウィンドウで表示させておくと、OSによるスリープを防げます。
  • カメラ権限: 起動時にカメラの使用を許可してください。

❓ よくある質問 (FAQ)

Q. アバターが動きません。

A. VRM1.0を使用し、送信機(Transmitter)の画面で送信開始をクリックしてみてください。また、受信機(Receiver)側で正しいPeerIDが入力されているかチェックしてください。

Q. 動作が非常に重いです。

A. ブラウザの「ハードウェアアクセラレーション」がオンになっているか確認してください（Chrome設定 ＞ システム）。
